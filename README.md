## Getting Started
### installation:
```
git init
git clone https://github.com/eddotbarbosa/vite-react-ts-api-consumer.git
npm install
```
### configs:
.env
```
API_URL="your API URL the default value is localhost:5000"
```
### running:
dev mode
```
npm run dev
```
build
```
npm run build
```
preview
```
npm run preview
```
